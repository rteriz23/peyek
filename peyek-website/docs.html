Documentation

Welcome to the official documentation for P.E.Y.E.K (Platform Extensible Yielding Engineering Kit). Whether you're integrating into a legacy CodeIgniter/Yii app or building a fresh React project, our SDK is designed to be completely framework-agnostic.

Installation (via CLI)

The fastest way to start a new project from scratch is using our CLI scaffolding tool. It automatically generates a boilerplate configured with our latest bundles.

npx create-peyek-app my-new-project

Follow the prompt to choose your framework (HTML, Laravel, Vue, etc).

CDN / Direct Script Inclusion

If you are injecting P.E.Y.E.K into an existing PHP application (like Laravel Blade or Yii View), you can simply include the built UMD bundles in your <head> tag:

<!-- Core Module --> <script src="https://unpkg.com/@rterizz23/peyek-core@latest/dist/peyek.core.min.js"></script> <!-- UI Builder Module (Optional) --> <script src="https://unpkg.com/@rterizz23/peyek-ui-builder@latest/dist/peyek.ui-builder.min.js"></script> <!-- PDF Viewer Module (Optional) --> <script src="https://unpkg.com/@rterizz23/peyek-pdf-viewer@latest/dist/peyek.pdf-viewer.min.js"></script>

Core Initialization

The core must be initialized before using the API tools, especially in secure environments like Laravel or Yii where CSRF tokens are strictly required. P.E.Y.E.K automatically scans your HTML meta tags for CSRF tokens.

// Standard Initialization PeyekCore.peyek.init(); // Premium SaaS Initialization (Requires valid license key from peyek.com) PeyekCore.peyek.init({ license: 'PYK-A1B2C3D4E5' });

Auto Modal

Generate beautiful, mobile-responsive modals via Javascript without needing HTML boilerplate or Bootstrap.

PeyekCore.Modal.show({ title: 'Payment Success', content: '<p>Your transaction was successful. Thank you!</p>', closeOnClickOutside: true });

API Wrapper

P.E.Y.E.K provides a lightweight fetch wrapper that automatically appends the framework's CSRF token to every POST request.

await PeyekCore.API.post('/api/save-user', { name: 'John Doe', role: 'Admin' });

UI Web Builder

Embed our drag-and-drop web builder into your application. Create an empty div container and pass its ID to the constructor.

<div id="my-builder"></div> <script> const builder = new PeyekUIBuilder.UIBuilder('my-builder'); builder.render(); </script>

PDF Viewer

Safely render PDFs using our optimized wrapper around Mozilla's pdf.js. The worker files are handled automatically underneath.

<div id="pdf-container" style="height: 600px;"></div> <script> const viewer = new PeyekPDFViewer.PDFViewer('pdf-container'); viewer.load('https://example.com/sample.pdf'); </script>
